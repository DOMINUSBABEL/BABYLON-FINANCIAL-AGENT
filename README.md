# BABYLON-FINANCIAL-AGENT
## Calculadora de Costos Internos y de Facturación/Cotización Externa

Este agente forma parte del ecosistema de **BABYLON.IA** y está diseñado bajo la **Arquitectura Geist** de automejora iterativa y la **Ley Cero de Asimov**. 

### 🌟 Propuesta de Valor Clave
* **100% Ejecución Local**: Absoluta confidencialidad de la información y tratamiento de datos propios de la empresa. La data es el activo más valioso; este software garantiza que ningún dato sensible se suba a nubes públicas de terceros.
* **Sin Límites de Tokens**: Al correr de manera local o integrada con modelos Open Source y puentes autorizados, opera sin restricciones de volumen de datos ni ventanas de bloqueo.
* **Enfoque B2B Pymes (Hasta 50 Trabajadores)**: Diseñado comercialmente para la venta estratégica en la temporada de presupuestos (Octubre-Noviembre).

### ⚙️ Funcionalidades
1. **Calculadora de Costos Internos**: Modelado dinámico de costos de personal (desarrollo, gestión), hardware local amortizado y costos de tokenización.
2. **Generador de Cotizaciones Externas**: Creación de propuestas comerciales bajo modelos de membresía anuales o soportes mensuales, calculando márgenes netos en tiempo real.
3. **Módulo de Telemetría Geist**: Registro en tiempo real de los ciclos dialécticos (Tesis, Antítesis y Síntesis) de viabilidad comercial del proyecto.

### 🚀 Instalación y Ejecución
1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Ejecuta el servidor local:
   ```bash
   npm start
   ```
3. Abre en tu navegador: `http://localhost:4001`
